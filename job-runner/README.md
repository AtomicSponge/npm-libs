#  job-runner

Run multiple processes simultaneously

Install to your existing project using:
```
npm i @spongex/job-runner
```

Include ECMAScript:
```
import { scriptError } from '@spongex/job-runner'
```

Include CommonJS:
```
const { scriptError } = require('@spongex/job-runner')
```
