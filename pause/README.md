#  pause

Pause for an amount of time in miliseconds

Install to your existing project using:
```
npm i @spongex/pause
```

Include ECMAScript:
```
import { pause } from '@spongex/pause'
```

Include CommonJS:
```
const { pause } = require('@spongex/pause')
```

# Usage

```
//  Pause for one second
await pause(1000)
```
